Detailed Description

The logical representation of the physical scale.

Member Function Documentation

◆ getAttributes()

abstract HashMap<String,Object> getAttributes ( )
abstract

Get the weight for the item on the scale, or null depending on the status.

If the scale is ready but does not have weight, this will return a 0 item weight, but the tare weight, decimal places, and unit of measurement may contain values depending on the configuration.
