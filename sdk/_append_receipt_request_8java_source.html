20  * <p>An API call to add data to the receipt footer. This can only be called within the context of a
21  * transaction, and should be called in response to a trigger before the transaction ends,
22  * generally in response to {@link com.verifone.commerce.triggers.PaymentCompletedNotification}.</p>
23  * <p>
24  * <p>This is currently abstract. in order to append to the receipt before this class is finally
25  * made concrete, please follow the instructions below in response to the
26  * {@link com.verifone.commerce.triggers.PaymentCompletedNotification PaymentCompletedNotification}.
27  * </p>
28  * <p>
29  * <p>When the notification is received, the app has 1-2 seconds to create the footer information
30  * and send it back. In the sample code below, the phrase "Have a nice day!" will be appended to
31  * the bottom of the receipt.
32  * {@code
33  * Intent intent = new Intent("com.verifone.commerce.intent.action.CP_APP_REQUESTS_APPEND_RECEIPT");
34  * intent.addCategory("com.verifone.commerce.intent.category.CP_MESSAGE");
35  * intent.putExtra("com.verifone.commerce.intent.extra.PAYLOAD", "<p>Have a nice day!</p>");
36  * context.sendBroadcast(intent);
37  * }</p>
38  * <p>
39  * <p>To append an image, the image data must be inserted inline to an HTML img tag using the data
40  * URL scheme, as defined at <a href="https://tools.ietf.org/html/rfc2397">
41  * https://tools.ietf.org/html/rfc2397</a>. More information can be found at
42  * <a href="https://en.wikipedia.org/wiki/Data_URI_scheme#HTML">
43  * https://en.wikipedia.org/wiki/Data_URI_scheme#HTML</a>, with an example below.
44  * {@code
45  * Intent intent = new Intent("com.verifone.commerce.intent.action.CP_APP_REQUESTS_APPEND_RECEIPT");
46  * intent.addCategory("com.verifone.commerce.intent.category.CP_MESSAGE");
47  * intent.putExtra("com.verifone.commerce.intent.extra.PAYLOAD", "<img src="data:image/png;base64,iVBORw0KGgoAAAANSUhEUgAAAAUAAAAFCAYAAACNbyblAAAAHElEQVQI12P4//8/* w38GIAXDIBKE0DHxgljNBAAO9TXL0Y4OHwAAAABJRU5ErkJggg==" alt="Red dot" />");
