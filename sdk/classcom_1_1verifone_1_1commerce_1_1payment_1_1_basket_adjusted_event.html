Detailed Description

When the basket is adjusted by applications in response to the SYSTEM_REQUESTS_BASKET_ADJUSTMENT Trigger, the final basket is returned in this event.

This allows the POS application to keep track of changes and provides the opportunity to finalize the basket in the response.
