8  * The base class for all peripherals with SDK-provided APIs. The statuses may be extended by the
9  * specific peripheral, meaning that {@link #getStatus() getStatus()} may return a String not
10  * identified in this parent class. The connection type may be more indicative of the Android
11  * connection rather than the peripheral. For example, it is possible that the Android will be using
12  * an ethernet cable to connect to the network that a peripheral is connected to. Even if the
13  * peripheral is using WiFi to connect, the Android may recognize it with the NETWORK connection
14  * type.
