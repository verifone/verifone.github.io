abstract static List<Scanner> getScanners ( )
staticabstract

Get the list of scanners as remembered by the device.

This may include scanners that are disconnected but not removed by the merchant.

Returns
A list of scanner objects. While this list contains references to the actual objects, the list itself is separate from the internally managed list. To be informed when new devices are connected or current devices are removed, please register a listener.
