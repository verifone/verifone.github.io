Detailed Description

Example to interact with a peripheral, in this case, a cash drawer.

Please note this class is built as an example, and therefore favors simplicity and accessibility over Android best practices.

void onStatusChanged ( Peripheral  peripheral,
String  status,
HashMap< String, Object >  attributes 
)

The PeripheralStatusListener method, to be notified when a specific peripheral changes status.

CashDrawer.STATUS_OPEN

indicates the drawer was opened, whether by the

open()

method or by some other method or application.

CashDrawer.STATUS_CLOSED

indicates the drawer has been closed.

For

Peripheral.STATUS_READY

, possibly save the peripheral to a local variable.

For

Peripheral.STATUS_BUSY

, possibly display an error to the user. This will not happen with a cash drawer, as it cannot be reserved.

For

Peripheral.STATUS_DISCONNECTED

, possibly display an error to the user.

For

Peripheral.STATUS_ERROR

, possibly display an error to the user.

For

Peripheral.STATUS_UNKNOWN

, this should probably be ignored, as future statuses without a proper adaptation to this version of the API will be labeled as this.
