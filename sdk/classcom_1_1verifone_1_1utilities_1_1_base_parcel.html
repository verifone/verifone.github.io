Detailed Description

A utility that provides some basic reflection-based parcel building.

Subclasses are expected to implement a constructor that receives a parcelable, and must call super when writing to the parcel. The ParcelCreator class should be used for the CREATOR object, this allows the object to be built properly from the parcel. This solves those problems associated with reconstructing objects that are sub-classes of the object defined in the AIDL.
