Detailed Description

The base class for all peripherals with SDK-provided APIs.

The statuses may be extended by the specific peripheral, meaning that getStatus() may return a String not identified in this parent class. The connection type may be more indicative of the Android connection rather than the peripheral. For example, it is possible that the Android will be using an ethernet cable to connect to the network that a peripheral is connected to. Even if the peripheral is using WiFi to connect, the Android may recognize it with the NETWORK connection type.
