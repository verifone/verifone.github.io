void onReceive (Context context, Intent intent)
 

Detailed Description

Example receiver for Commerce Platform Triggers.

This class is built as an example and is not compilable source code, but is meant to provide direction. It therefore favors simplicity and accessibility over Android best practices.

See also
Please refer to the Android documentation at http://developer.android.com/reference/android/content/BroadcastReceiver.html for further information.

Generating a Commerce Message object from the intent. The builder will always return the appropriate concrete class for the intent.

CommerceMessageBuilder builder = new CommerceMessage.CommerceMessageBuilder();
CommerceMessage message = builder.buildMessageFromIntent(intent);

Starting a service within the onReceive. Assumes ServiceExample is a registered service.

Intent intent = new Intent(intent);
intent.setComponent(ServiceExample.class);
context.startService(intent);

Binding to a service within the onReceive. Assumes ServiceExample is a registered service.

Intent intent = new Intent(intent);
intent.setComponent(ServiceExample.class);
IBinder binder = this.peekService(context, intent);

Starting an activity within the onReceive. Assumes ActivityExample is a registered activity.

Intent intent = new Intent(intent);
intent.setComponent(ActivityExample.class);
intent.setFlags(Intent.FLAG_ACTIVITY_NEW_TASK);
context.startActivity(intent);
