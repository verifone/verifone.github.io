Detailed Description

Allows applications to enable/disable accessibility options for the customer experience. Accessibility controls for the merchant/cashier experience can be found in the main system settings.

◆ ACTION_ACCESSIBILITY_CHANGED

final String ACTION_ACCESSIBILITY_CHANGED = "com.verifone.device.intent.action.ACCESSIBILITY_CHANGED"
static

The intent action for a broadcast that indicates the accessibility mode has changed on the device. There are a few places that can change the accessibility mode, including the customer-facing screen. This broadcast is fired whenever the mode is changed, so it is best to register a dynamic receiver only when the application cares that it might change. This should not be registered in the manifest.
